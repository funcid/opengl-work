=======================================================================
Исполняемые Файлы Проектов по Компьютерной Графике с OpenGL
=======================================================================

Эта папка содержит автономные исполняемые файлы для различных проектов 
по компьютерной графике. Каждый файл можно запустить независимо, 
не требуется установка Python или дополнительных библиотек.

Проекты:
--------
1. project1_bin.exe - Платонические тела (куб, тетраэдр, октаэдр, икосаэдр, додекаэдр)
2. project2_bin.exe - Алгоритмы рисования линий (Брезенхем, ЦДА)
3. project3_bin.exe - Алгоритмы отсечения линий
4. project4_bin.exe - Цветовые системы (RGB, CMYK, HSV, LAB)
5. project5_bin.exe - Алгоритмы дизеринга изображений
6. project6_bin.exe - Растеризация эллипсов

Использование:
-------------
1. Дважды щелкните на файле, чтобы запустить его напрямую.
2. Для выхода из программы нажмите ESC или закройте окно.

Примечания:
----------
- Эти исполняемые файлы оптимизированы по размеру с использованием:
  * Исключения неиспользуемых модулей Python
  * Удаления отладочных символов
  * Сжатия UPX (если доступно)
  * Индивидуальных спецификаций сборки
- Если программа не запускается, попробуйте запустить её с правами 
  администратора.

Управление:
----------
- WASD: перемещение камеры
- Мышь: вращение камеры
- Колесо мыши: масштабирование
- ESC: выход из приложения
- Дополнительные клавиши описаны внутри каждого приложения


=======================================================================
OpenGL Graphics Projects - Executable Binaries
=======================================================================

This folder contains standalone executable files for various computer 
graphics projects. Each file can be run independently, without 
requiring Python or additional libraries to be installed.

Projects:
--------
1. project1_bin.exe - Platonic solids (cube, tetrahedron, octahedron, icosahedron, dodecahedron)
2. project2_bin.exe - Line drawing algorithms (Bresenham, DDA)
3. project3_bin.exe - Line clipping algorithms
4. project4_bin.exe - Color systems (RGB, CMYK, HSV, LAB)
5. project5_bin.exe - Image dithering algorithms
6. project6_bin.exe - Ellipse rasterization

Usage:
-----
1. Double-click on a file to run it directly.
2. To exit the program, press ESC or close the window.

Notes:
-----
- These executable files are size-optimized using:
  * Exclusion of unused Python modules
  * Removal of debug symbols
  * UPX compression (if available)
  * Individual build specifications
- If a program doesn't start, try running it with administrator privileges.

Controls:
-------
- WASD: camera movement
- Mouse: camera rotation
- Mouse wheel: zoom
- ESC: exit application
- Additional keys are described within each application
